To compile this program:

1. Run the command: "make"

To run this program:
2. Run: "./dir_traversal <directory name>"
3. If you'd like to traverse the entire path you're at (and not a specific directory), run: "./dir_traversal"

If you'd like to compile and run it at once, run: "make && sudo ./dir_traversal <directory name>" or if you wish for the same as step 3, run: "make && ./dir_traversal"
If you'd like to erease compiled files run the command: "make clean"
