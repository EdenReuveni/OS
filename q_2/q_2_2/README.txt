To compile this program:

1. Run the command: "make"

To run this program:
2. Run: "./server"
3. In a different terminal (but on the same directory) run: "./client <serever pid><signal type><amount of signals>" with the pid printed in the first terminal you opened

If you'd like to compile and run it at once, run the command: "make && ./server" and go back to step 3
If you'd like to erease compiled files run the command: "make clean"
