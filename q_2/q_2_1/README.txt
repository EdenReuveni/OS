To compile this program:

1. Run the command: "make"

To run this program:
2. Run the command: "ps -A" to show all running process and their pids (or a different command of your choice)
3. Run: "./check_pid <pid>" with a pid from step 2 or with one that's not shown

If you'd like to compile and run it at once- after doing step 2, run the command: "make && ./check_pid <pid>"
If you'd like to erease compiled files run the command: "make clean"
